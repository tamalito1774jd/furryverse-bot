# FurryVerse Bot - Bienvenida 🎉

¡Hola! Has clonado exitosamente **FurryVerse Bot**, un bot Discord profesional multipropósito.

## 🚀 Iniciar Rápidamente

### 1️⃣ **Instalación**
```bash
npm install
cd bot && npm install
cd ../backend && npm install
cd ../dashboard && npm install
cd ..
```

### 2️⃣ **Configuración**
```bash
cp .env.example .env
# Edita .env con tus credenciales Discord
```

### 3️⃣ **Desarrollo Local**
```bash
npm run dev
```

### 4️⃣ **Producción (Docker)**
```bash
docker-compose up -d
```
